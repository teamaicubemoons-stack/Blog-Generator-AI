# 🚀 AI Blog Generator Agent (9.7+ Authority System)

An advanced, multi-agent AI system designed to generate high-authority, SEO-optimized, and conversion-focused B2B blog posts. Built with **FastAPI**, **Groq (Llama 3.3 70B)**, and **React**.

## 🌟 Key Features
- **9.7+/10 Authority Quality**: Surgical content refinement for board-room ready output.
- **Multi-Agent Orchestration**: Specialized agents for Trends, Keywords, SEO Planning, and Generation.
- **Technical SEO Excellence**: Automatic JSON-LD FAQ Schema generation and semantic keyword variation.
- **E-E-A-T Optimized**: Natural integration of external authority signals (Gartner, McKinsey) and soft-qualifiers for ROI claims.
- **Consultant-Level Tone**: Narrative-driven storytelling with "Day in the Life" segments and expert PRO TIP boxes.
- **Conversion Focused**: ROI-driven CTAs with measurable outcomes and subtle lead magnet integration.

## 🏗️ Architecture
The system uses a phased pipeline:
1. **Trend Analysis**: Scrapes real-time data for gaps and competitor strategies.
2. **Keyword Generation**: Identifies primary, secondary, and LSI keywords.
3. **SEO Structure Planning**: Maps the H1-H3 hierarchy with a focus on "Insight Density."
4. **Content Generation**: Produces 2,500+ words of granular, non-fluffy content.
5. **SEO Optimization**: Refines readability and keyword density while preserving tone.

## 🛠️ Setup & Installation

### Backend (Python)
1. Navigate to `/backend`
2. Create a virtual environment: `python -m venv .venv`
3. Activate it: `.\.venv\Scripts\activate` (Windows) or `source .venv/bin/activate` (Mac/Linux)
4. Install dependencies: `pip install -r requirements.txt`
5. Configure `.env` with your `GROQ_API_KEY` and `SERPER_API_KEY`.
6. Run the server: `python main.py`

### Frontend (React/Vite)
1. Navigate to `/frontend`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
